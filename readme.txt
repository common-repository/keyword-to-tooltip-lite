Easily add a brief to long explanation to certain words all over your BLOG

== Description ==

Certain words simply need to be explained, that's why we've created this plugin

Why you should consider Keyword to Tooltip

Keyword to Tooltip is the best way to easily attach tooltips to your most "ambiguous" words or word

Keyword to Tooltip is the best solution to easily attach a brief description to technical terms in seconds

1. WordPress Multi Site Support
1. Easily control all your tooltips from ONE place !
1. Intuitive Administration System

== Installation ==

1. Upload `keyword-to-tooltip-lite` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find the "KTool Lite" icon in your left menu, and start "rocking"
